맞춤형 취업 로드맵 설계 서비스 '스텝업(Step-Up)' 기획안
1. 서비스 컨셉 및 목표
서비스명: 스텝업 (Step-Up)
슬로건: 당신의 스펙, 목표를 향한 가장 스마트한 발걸음
핵심 컨셉: 사용자의 현재 스펙을 객관적으로 분석하고, 실제 기업의 채용 공고를 기반으로 한 맞춤형 목표를 설정하여, 달성을 위한 구체적인 일정과 업무를 제공하는 개인 맞춤형 취업 로드맵 관리 웹 서비스
서비스 목표:
취업 준비생들이 막연하게 스펙을 쌓는 것이 아닌, 명확한 목표를 가지고 체계적으로 준비할 수 있도록 돕는다.
자신의 현재 위치를 시각적으로 확인하고, 목표와의 격차를 인지시켜 학습 동기를 부여한다.
복잡한 취업 준비 과정을 하나의 서비스 안에서 통합 관리하여 편의성을 제공한다.
2. 주요 기능 및 화면 구성
가. 회원가입 및 로그인
기능:
아이디, 비밀번호, 이메일 주소를 이용한 간단한 회원가입 기능
아이디와 비밀번호를 통한 로그인
아이디/비밀번호 찾기 기능
나. 최초 정보 입력 (온보딩 프로세스)
흐름: 로그인 후, 최초 1회에 한해 자신의 상세 스펙을 입력하는 페이지로 이동합니다.
입력 항목 (카테고리별 분리):
기본 정보: 이메일 주소(자동 입력), 희망 직무 분야(예: 프론트엔드 개발, 마케팅, 기획)
학력: 학교, 전공, 학점, 졸업(예정) 여부
어학 능력: 공인 어학 성적 (종류, 점수, 취득일)
자격증: 보유 자격증명, 취득일
대외활동: 인턴, 공모전, 봉사활동, 동아리 등 활동 내용 및 기간
프로젝트 경험: 참여 프로젝트명, 역할, 기간, 사용 기술/툴, 성과 (GitHub, 포트폴리오 링크 첨부 가능)
자기소개: 자신을 어필할 수 있는 핵심 역량 및 경험을 간략히 기술
다. 내 스펙 분석 대시보드
기능: 입력된 스펙 정보를 바탕으로 현재 상태를 시각적으로 보여주는 개인화된 대시보드입니다.
화면 구성:
종합 스펙 요약: 사용자의 스펙을 '학력', '어학', '경험', '역량' 등 카테고리별로 나누어 레이더 차트(방사형 차트)나 막대그래프로 시각화하여 강점과 약점을 한눈에 파악할 수 있도록 제공합니다.
스펙 리스트: 입력한 모든 스펙을 목록 형태로 깔끔하게 정리하여 보여줍니다. (수정 및 삭제 가능)
'목표 설정하기' 버튼: 스펙을 보강하기 위한 다음 단계(목표 설정)로 넘어가는 버튼을 배치합니다.
라. 목표 설정 (로드맵 설계)
기능: 실제 기업들의 채용 공고를 기반으로 자신의 목표를 설정하고, 현재 스펙과의 차이점(Gap)을 분석합니다.
화면 흐름:
채용 공고 탐색: 사용자가 희망하는 기업명이나 직무 키워드로 실제 채용 공고를 검색할 수 있습니다. (외부 채용 사이트 정보 연동)
목표 공고 선택: 마음에 드는 채용 공고를 '내 목표로 설정하기' 버튼을 눌러 선택합니다.
스펙 비교 분석: 선택한 공고의 자격요건/우대사항과 사용자의 현재 스펙을 자동으로 비교 분석하여 부족한 부분을 명확하게 보여줍니다.
예시)
[필요 스펙] 정보처리기사 자격증 보유 → [내 스펙] 미보유 (❌)
[필요 스펙] TOEIC 850점 이상 → [내 스펙] 780점 (⚠️ 부족)
[필요 스펙] 프로젝트 경험 1회 이상 → [내 스펙] 프로젝트 경험 2회 (✅ 충족)
보완할 업무 리스트업: 분석된 '부족한 스펙'들을 기반으로 달성해야 할 구체적인 업무(Task) 리스트를 자동으로 추천해 줍니다.
예시) 정보처리기사 자격증 취득하기, TOEIC 850점 달성하기, 포트폴리오 사이트 만들기 등
마. 메인 화면 (로드맵 진행 관리)
기능: 설정한 목표와 생성된 업무 리스트를 바탕으로 현재 진행 상황을 실시간으로 확인하고 관리하는 메인 페이지입니다.
화면 구성:
목표 정보: 현재 목표로 설정한 '회사명 - 직무'가 상단에 표시됩니다.
종합 진행도: 목표 달성까지의 전체 진행률을 퍼센트(%)와 원형 그래프로 시각적으로 보여줍니다.
오늘의 할 일 (To-do List): 생성된 업무 리스트 중 오늘 하기로 설정된 일들이 표시됩니다. 완료 시 체크하여 진행도에 즉시 반영됩니다.
전체 업무 관리: 생성된 모든 업무를 캘린더 형태나 칸반 보드 형태로 볼 수 있으며, 사용자가 직접 일정을 수정하거나 새로운 업무를 추가할 수 있습니다.
D-Day: 목표 공고의 마감일까지 남은 날짜를 보여주어 긴장감을 유지시킵니다.
3. 사용자 시나리오 (User Flow)
신규 사용자 A는 '스텝업' 서비스에 접속하여 회원가입을 한다.
로그인 후, 안내에 따라 자신의 학력, 어학 점수, 프로젝트 경험 등 스펙을 상세히 입력한다.
**'내 스펙 분석 대시보드'**에서 자신의 강점과 약점을 시각적인 차트로 확인한다.
'목표 설정하기' 버튼을 눌러 평소 가고 싶었던 'B회사'의 '신입 마케터' 채용 공고를 검색하고 목표로 설정한다.
서비스는 A의 스펙과 B회사 공고의 요구사항을 자동으로 비교하여 'GAIQ 자격증 미보유', 'SNS 채널 운영 경험 부족'이라는 격차(Gap)를 보여준다.
이 격차를 바탕으로 '[GAIQ 자격증 취득]', '[개인 블로그 개설 및 운영]' 등의 업무(Task)가 자동으로 생성된다.
A는 생성된 업무들의 일정을 조율하고, 메인 화면으로 돌아온다.
이후 A는 매일 메인 화면에 접속하여 '오늘의 할 일'을 확인하고 완료하며, 목표를 향한 진행도가 실시간으로 올라가는 것을 보며 동기부여를 얻는다.
